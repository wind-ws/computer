
多空 与 看涨看跌 无关


| 仓位                   | 产品   | 盈利方向 | 实际含义 |
| -------------------- | ---- | ---- | ---- |
| **认购多头（Long Call）**  | Call | 上涨赚钱 | 看涨   |
| **认购空头（Short Call）** | Call | 下跌赚钱 | 看跌   |
| **认沽多头（Long Put）**   | Put  | 下跌赚钱 | 看跌   |
| **认沽空头（Short Put）**  | Put  | 上涨赚钱 | 看涨   |

“多头 / 空头”
表示 **你是买方还是卖方**：
- **多头** = 买方（你付钱买权利或多仓）
- **空头** = 卖方（你收钱卖权利或开空）

 “看涨 / 看跌”
表示 **你押价格走向哪边**：
- **看涨** = 价格上涨你赚钱
- **看跌** = 价格下跌你赚钱
