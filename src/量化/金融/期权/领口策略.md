
领口策略,就是在持有[[src/量化/金融/期权/保险策略|保险策略]]的同时,卖出认购期权,以认购期权所得的权利金收入降低整个保险策略的成本

领口策略的构建成本低于保险策略,但组合收益有上限
最大损失 = (标的证劵购买价格 - 认沽期权行权价) + (认沽期权权利金 - 认购期权权利金)
盈亏平衡点 = 购买标的证劵价格 + 认沽期权权利金 - 认购期权权利金
最大收益 = (认购期权行权价 - 标的证劵购买价格) + (认购期权权利金 - 认沽期权权利金)
