
期权是一种选择 是否交易的权利
其中 
认购期权是 买入权利
认沽期权是 卖出权利

期权由以下概念组成:
到期日,
行权价,
权利金: 买入期权的价格
...

美式期权 是 一种到期前 仍和一天都可以行使权力的 期权
欧式期权 是 一种只能在到期日当天行使权力的 期权

到期月份 是 当月第四个星期三

在刚刚购买期权后 立刻执行权利, 赚钱则是 **实值(价内)** , 不赚不亏则为 **平值(平价)**, 亏钱则为**虚值(价外)** 

期权的权利金 总价值 = 权利金的内在价值 + 权利金的时间价值
[期权的价值](https://zh.wikipedia.org/wiki/%E6%9C%9F%E6%AC%8A#%E9%81%B8%E6%93%87%E6%AC%8A%E7%9A%84%E5%83%B9%E5%80%BC)

| 影响因素  | 认购期权 | 认沽期权 |                         |
| ----- | ---- | ---- | ----------------------- |
| 标的价格  | +    | -    | 标的价格越高 认购期权价格就越高        |
| 时间    | +    | +    | 到期时间越长 期权价格越高           |
| 波动率   | +    | +    | 波动率越大 期权价格越高            |
| 行权价格  | -    | +    | 相比于标的价格,行权价格越高 认购期权价格越低 |
| 无风险利率 | +    | -    | #todo                   |
| 股息率   | -    | +    | #todo                   |

期权交易的6个买卖类型:

| 类型         | 保证金        | 仓位变动    |
| ---------- | ---------- | ------- |
| 买入开仓       | 无          | 权利仓增加   |
| 卖出平仓       | 无          | 权利仓减少   |
| 卖出开仓       | 需缴保证金      | 义务仓增加   |
| 买入平仓       | 成交后释放保证金   | 义务仓减少   |
| 备兑开仓(策略指令) | 需要100%现卷担保 | 备兑义务仓增加 |
| 备兑平仓(策略指令) | 成交后释放担保现卷  | 备兑义务仓减少 |
例如:
以0.15(权利金)的价格买入 认购期权(即买入开仓), 后 以 0.18(权利金)的价格卖出 认购期权(即卖出平仓), 则盈利 $(0.18/0.15) -1$ 
对于卖出开仓, 以 0.15(权利金)的价格卖出 认购期权(即卖出开仓),卖出后立马获得 权利金 $0.15*n$ , 后以 0.12(权利金)的价格买入 认购期权(即买入平仓) , 付出 权利金 $0.12 * n$, 则盈利 $(0.15 / 0.12) -1$


### 买方风险
1. 价值归零风险
2. 高溢价风险
3. 流动性风险
4. 行权交割风险
   一是 到期日 忘记行权
   二是 对于认购期权的买方,若在T日行权,必须等到 T+1 日才能得到劵, T+2日才能卖出标的证劵. 这就带来了两天的价格波动风险

### 卖方风险
1. 保证金风险
   若保证金不足 会强制平仓
   开仓保证金: 是开仓时需要缴纳的保证金
   维持保证金: 是用来维持仓位的保证金, 让仓位不被强平
2. 巨额亏损风险
3. 流动性风险
4. 行权交割风险
   对于卖方 存在一个交割违约风险
   若买方在T日行权后,卖方需要在T+1日把资金或标的证劵交割个买方,若没有做,则构成违约.
